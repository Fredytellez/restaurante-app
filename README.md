# 🍽️ Restaurante App

Sistema de gestión para restaurantes y café bares con experiencia digital para el cliente y dashboard en tiempo real para el negocio.

## ✨ Funcionalidades

- 📱 Página del cliente via código QR por mesa
- 🛒 Pedidos en tiempo real
- 🔔 Solicitud de mesero, cuenta y limpieza
- 🎵 Solicitud de música
- 📋 Dashboard para gestión de mesas y meseros

## 🛠️ Stack tecnológico

- **Backend:** Node.js, Express, Socket.io
- **Frontend:** Next.js
- **Base de datos:** PostgreSQL

## ⚙️ Instalación

### 1. Clonar el repositorio
```bash
git clone 
cd restaurante-app
```

### 2. Configurar el backend
```bash
cd backend
npm install
```

```bash
npm run dev
```

### 3. Configurar el frontend
```bash
cd frontend
npm install
npm run dev
```
